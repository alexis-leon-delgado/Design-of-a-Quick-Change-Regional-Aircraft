# Design-of-a-Quick-Change-Regional-Aircraft
Preliminary design of a quick-convertible aircraft between passengers and cargo operational modes specifically conceived to operate in short regional range flights and following both EASA and FAA regulations.
Acted as the main project coordinator, and achieved "First Class Honours" in the associated BSc PM module.
